# Whoami Quiz 企画（最小構成版）

## 目的

自分のパーソナル情報をAIに読ませ、AIがその人を表す5問クイズを作る。
友人がWeb上で回答し、正答数に応じて「その人をどれくらい理解しているか」を確認できる。

## MVPの方針

- バックエンドなしの静的Webアプリとして作る
- AI連携は手動運用（プロンプトをコピー → AIに渡す → 生成JSONを貼り付け）
- 設問は **5問固定**
- 1ページ内で「作問モード」と「回答モード」を切り替える

この割り切りにより、APIキー管理なしで即運用でき、作問者がAI出力を確認してから公開できる。

## 運用フロー

1. 作問者がパーソナル情報を入力する
2. アプリがAI用プロンプトを生成する
3. 作問者がプロンプトをコピーしてAIに渡す
4. AIが5問の4択クイズJSONを生成する
5. 作問者がJSONを貼り付けて読み込む
6. 友人が回答する
7. 採点して正答数と理解度メッセージを表示する

## 画面構成

1ページ構成。作問モードと回答モードをトグルで切り替える。

### 作問モード（作問者向け）

- パーソナル情報の入力（textarea）
- AI用プロンプトの生成＆コピー
- 生成された設問JSONの貼り付け＆読み込み

### 回答モード（友人向け）

- クイズタイトルの表示
- 5問の4択表示と回答選択
- 採点
- 正答数と理解度メッセージの表示
- 各問題の正解と補足説明の表示
- もう一度回答

## データ構造

AIが生成する設問JSONは以下の形式。`questions` は5件。

```json
{
  "title": "ユウスケ理解度チェック",
  "questions": [
    {
      "question": "よく言いそうな一言は？",
      "choices": [
        "まず最小で試そう",
        "全部あとで",
        "会議を増やそう",
        "気合いだけで進めよう"
      ],
      "answerIndex": 0,
      "explanation": "最小実装から試す傾向があります。"
    }
  ]
}
```

### フィールド定義

| フィールド | 型 | 内容 |
|---|---|---|
| `title` | string | クイズタイトル |
| `questions` | array | 設問一覧（5件） |
| `question` | string | 設問文 |
| `choices` | string[] | 4択の選択肢 |
| `answerIndex` | number | 正解選択肢の番号。0始まり |
| `explanation` | string | 正解発表時の補足説明 |

## 採点ロジック

回答者が選択した選択肢の番号と、各設問の `answerIndex` を比較する。

```text
正解数 = answerIndex と選択値が一致した問題数（0〜5）
```

## 理解度メッセージ

設問が5問固定なので、正解数で判定する。

| 正解数 | 表示文言 |
|---|---|
| 5問 | 完璧！あなたはストーカーですか？ |
| 4問 | かなり分かってる |
| 3問 | 友人として順調 |
| 1〜2問 | まだ伸びしろあり |
| 0問 | はじめまして級 |

## バリデーション

設問JSON読み込み時に以下を検証する。

- JSONとしてパースできること
- `title` が空でない文字列であること
- `questions` がちょうど5件の配列であること
- 各設問に `question` があること
- `choices` が4件あること
- `answerIndex` が0〜3の整数であること

## 永続化

読み込んだクイズは `localStorage` に保存し、再読み込みで復元する。

保存キー:

```text
whoami.quiz.v1
```

## AIプロンプトの方針

AIには以下を指示する。

- 設問は5問にする
- 友人向けに楽しく解ける難易度にする
- 事実確認だけでなく、価値観・口ぐせ・行動パターンも設問化する
- 選択肢は4つにする
- 正解番号は `answerIndex` に0始まりで入れる
- 補足説明を `explanation` に入れる
- 出力はJSONのみとする

## MVPで省略する機能

シンプルさを優先し、初期版では以下を作らない。

- 設問JSONの書き出し
- サンプルデータの読み込み
- クイズURLの共有
- 複数人分のクイズ保存
- 回答者名の入力・回答履歴・ランキング
