# Whoami Quiz

パーソナル情報をAIに読ませて、友人向けの4択クイズとして出題する静的Webアプリです。

## 使い方

1. `index.html` をブラウザで開く
2. 「作問」でパーソナル情報を入力する
3. 「プロンプト作成」でAI用プロンプトを作る
4. AIが返したJSONを「設問JSON」に貼り付けて読み込む
5. 「回答」でクイズを解く

## 設問JSON

```json
{
  "title": "ユウスケ理解度チェック",
  "questions": [
    {
      "question": "よく言いそうな一言は？",
      "choices": ["まず最小で試そう", "全部あとで", "会議を増やそう", "気合いだけで進めよう"],
      "answerIndex": 0,
      "explanation": "最小実装から試す傾向があります。"
    }
  ]
}
```
